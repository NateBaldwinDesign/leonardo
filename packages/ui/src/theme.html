Create beautiful, accessible colors

First of its kind contrast-ratio based, fully adaptive color theme generator

Leonardo generates colors based on desired contrast ratios. It allows you to create parameters for each color so that you can control the hue and saturation changes for color as it gets lighter or darker.

Designers and engineers can both collaborate on this tool, as the parameters defined here can be passed into the leonardo npm module to generate colors during a build or dynamically in your product.

The latter of those implementations enables innovative methods for your product to achieve dynamic accessibility, providing your customers with a wider array of color and contrast needs in their control.
